Microsoft dá pendrive grátis para quem quer testar o Windows 11 beta

A Microsoft anunciou quarta-feira (15) uma novidade para facilitar a vida dos Insiders, aqueles usuários avançados que testam as últimas novidades do Windows e, eventualmente, verificam bugs antes do lançamento para o público geral. A empresa está enviando unidades USB gratuita para os que desejarem experimentar a mais nova versão “Canary” do Windows 11.

A ideia é que os usuários inscritos no programa Canary possam fazer uma instalação limpa do Windows 11 e começar a usar as compilações diárias da versão mais instável e menos testada do Windows Insider.

A novidade ocorre após a companhia de Redmond ter introduzido, no início do mês, um novo Canary Channel. Embora a notícia tenha causado frisson entre aqueles que adoram experimentar a vanguarda do software, a necessidade instalar o Windows do zero afastou aqueles que não dispunham de um drive exclusivo para esse fim.

A parte um pouco triste desta notícia é que não existe um caminho para você pedir o seu drive USB gratuito. De acordo com o Beta News, a Microsoft é quem tomará a iniciativa de enviar um convite por e-mail — com um código de verificação — para permitir que a pessoa contemplada faça a solicitação.

Segundo a publicação, o correio eletrônico chega com o seguinte texto: "Para realizar uma instalação limpa do Windows, recomendamos o uso de uma unidade flash USB em branco com pelo menos 8 GB de armazenamento. Se você não tiver um, solicite um USB usando este código: XXXXXXXXXX. Fornecimento limitado. Os itens estão disponíveis por ordem de chegada".

Como o estoque é limitado, a Microsoft alerta aos que receberem o e-mail que façam rapidamente seus pedidos. O envio pode levar de seis a oito semanas.
